# 使用面包版DIY制作一个arduino/DIY an arduino using a breadboard

**前言/Foreword**

这个帖子将事无巨细地告诉大家如何用几个零件和一个面包版做一个arduino，这样就不需要购买一个arduino。同时，帖子也会包含零件的基本介绍，也将会更加深度地了解电路板的构造。
